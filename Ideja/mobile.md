# Trainwatchers

Android aplikacija za popisivanje vlakova kojima je korisnik putovao ili koje je vidio.

## Struktura aplikacije

Start screen
- welcome
- statistika - imate X vlakova na svom popisu
- pregled posljednje dodanih vlakova

Add new
- dodavanje novog vlaka

Trains
- popis svih unešenih vlakova
- klikom na neki se otvara novi screen koji prikazuje više informacija
